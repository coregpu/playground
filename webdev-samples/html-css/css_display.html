This div is hidden by default using display:none.

When user clicks the above panel the div is shown using display:block. The next click will set the panel's display attribute to none and so on.

This is accomplished by binding a Javascript function to the onclick even of the above div. When the user clicks the above div, this function will toggle the state of the display attribute of this panel (will set it to display:block if it is currently set to display:none or to display:none if it is currently set to display:block).
